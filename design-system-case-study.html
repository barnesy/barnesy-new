The Challenge

When I took ownership of the design system, we needed more than incremental improvements. The existing system couldn't support our velocity, and the design-to-code gap slowed the entire product development cycle.

The vision: rebuild the design system to enable AI-powered workflows, achieve 100% pattern coverage, and generate production-ready code. A foundation for a new way of building products.

Agent Studio dashboard showing the AI-powered design system in action

The Approach

1

Rebuild with MUI React + Design Tokens

Rebuilt the Figma design system using Material UI React as the foundation. Integrated design tokens for colors, typography, spacing, and all system primitives. Created a single source of truth aligned with production code.

2

Test Code Generation with Figma MCP

Tested code generation using Figma's Model Context Protocol (MCP). Proved automated design-to-code was viable with significant performance improvements. Validated the system could power AI-driven workflows.

3

Create Patterns Figma

Built Patterns Figma, a comprehensive pattern library achieving 100% coverage of common interface needs. Each pattern structured with semantic naming and consistent properties that both humans and AI systems can understand.

Patterns Figma library showing comprehensive component coverage

4

Build Seamstress: AI Prototype Generator

Created Seamstress, an AI agent that generates pixel-perfect prototypes directly from Patterns Figma. Closed the design-to-code gap‚Äîdescribe what you need, get production-ready code using exact design system patterns.

5

Design Agent Studio with Generative AI

Ultimate test: designing Agent Studio, a new agent and context management system. 80% of designs generated using AI workflows, proving the system powers real product development at scale.
