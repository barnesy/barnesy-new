## Deployment

### Option 1: Direct Upload
Upload `index.html` to your web host at barnesy.me

### Option 2: GitHub Pages
```bash
git init
git add index.html README.md
git commit -m "AI-focused redesign with CDS 2.0 showcase"
git branch -M main
git remote add origin <your-repo>
git push -u origin main
```
Then enable GitHub Pages in repository settings.

### Option 3: Netlify/Vercel
Drag and drop the folder to Netlify or Vercel for instant deployment.
